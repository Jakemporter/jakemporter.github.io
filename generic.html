Actualize Bootcamp Capstone Project

I took a full-stack coding bootcamp for about 5 months in 2020. It was online based and I took it during the evenings because I had work. I learned Ruby on Rails on the backend and Vue.js on the frontend. The last month or so of the bootcamp we were tasked to create a website from scratch. Frontend and backend was written by me. I have an interest in cars and decided to make a car auction website where you could view a list of cars on the main page and click on different cars, make bids, comments etc. The website also connected to an external API that used the VIN of the car to get current auction values and a plethora of other information. This is a link to a Youtube video where I presented this project.
